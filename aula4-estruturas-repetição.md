# Estruturas de Repetição na Lógica de Programação

As **estruturas de repetição**, também conhecidas como **loops** ou **laços de repetição**, são recursos fundamentais na lógica de programação. Elas permitem que um bloco de código seja executado repetidamente enquanto uma determinada condição for verdadeira ou por um número definido de vezes.

O principal objetivo é evitar a redundância de código e facilitar a automação de tarefas iterativas.

---

## 1. O Laço `Enquanto` (While)

O laço `While` é utilizado quando **não sabemos exatamente quantas vezes** o código precisará ser repetido, mas sabemos a condição que deve ser mantida para que a repetição continue.

### Funcionamento:
1. A condição é verificada **antes** de entrar no laço.
2. Se a condição for verdadeira, o bloco de código é executado.
3. Após a execução, a condição é testada novamente.
